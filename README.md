# Pinball Front End
Pinball Front End is a high speed interface for pinball cabinets that allows launching of different digital pinball emulators / software.

## Default Keybindings:
| Function | Keybind |
| :----------- | :----- |
| Table Manager | F1 |
| Exit Table Manager | Escape |
| Random Table | F2 |
| Next Table | Right |
| Previous Table | Left |
| Start Table | Enter |
| Exit Table | Escape |
| Exit Front End | Escape |
